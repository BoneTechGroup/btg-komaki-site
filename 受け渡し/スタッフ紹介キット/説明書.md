# スタッフ紹介セクション 移植キット

交通事故サイト（btg-komaki-hp）で好評だった「スタッフ紹介」セクションを、
この公式サイトに移植するためのキットです。**このフォルダだけで完結**します。

## デザインの特徴（これを再現する）
- 実写写真を**ポラロイド風**（白い太フチ・少し傾き）で表示
- 写真の足元に**ちびキャラのイラストを重ねて**配置（岩井=右下、村瀬=左下）
- ホバーで写真の傾きが戻り、キャラがぴょこっと浮く
- スクロールで左右からふわっと出現（reveal アニメ）

## 同梱ファイル
| ファイル | 内容 |
|---|---|
| `スタッフ紹介.html` | セクションのHTML断片＋出現アニメ用の最小JS |
| `スタッフ紹介.css` | 必要なCSS一式（自己完結。CSS変数のフォールバック付き） |
| `画像/staff-iwai.jpg` | 岩井 拓磨の実写（装飾文字入り） |
| `画像/staff-murase.jpg` | 村瀬 将之の実写（装飾文字入り） |
| `画像/iwai3.png` | 岩井のちびキャラ（透過PNG） |
| `画像/murase1.png` | 村瀬のちびキャラ（透過PNG） |

## 移植手順
1. `画像/` の4ファイルを公式サイトの画像フォルダ（例: `写真/` や `assets/`）へコピー
2. `スタッフ紹介.html` のセクションを設置したい場所に貼り付け、`<img src>` のパスを手順1の場所に合わせて修正
3. `スタッフ紹介.css` を既存CSSに統合
   - 公式サイトに同名のCSS変数（--tiffany等）が既にあれば、冒頭の `:root` ブロックは削除
   - セクション見出し（.sec-head）の仕組みが既にあるなら、そのブロックも不要
   - **テーマカラーは公式サイトの配色に合わせて変更してOK**（--tiffany を差し替えるだけ）
4. スクロール出現アニメ:
   - 公式サイトに IntersectionObserver で `.reveal` に `.in` を付ける仕組みが既にあれば、HTML末尾の `<script>` は削除
   - なければそのまま使う

## 注意
- ちびキャラPNGは大きい（2MB超）ので、必要なら幅600px程度に縮小して軽量化推奨
- 写真に焼き込まれた手書き文字（趣味など）は画像の一部。文字を変えたい場合は元データが必要なので発注者に確認
- キャラの白フチ・水彩タッチはブランド共通の世界観。別スタッフを追加する場合は同じテイストで生成すること
