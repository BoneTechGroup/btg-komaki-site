# BTG接骨院 小牧院 公式サイト リメイク

現行サイト https://sekkotsuin-komaki.com のデザインリニューアル作業用リポジトリ。

## 構成
- `index.html` — トップページのデザイン案（フルスクリーン動画ヒーロー / パララックス / オリジナルSVGアイコン）
- `css/style.css` — スタイル一式（テーマカラー: ティファニーブルー #0abab5）
- `写真/` — 撮影素材の元データ（施術写真・動画）
- `写真/web/` — Web用に軽量化した画像(1600px)と動画(1280px mp4)

## メモ
- ブラウザで index.html を開けばそのままプレビュー可能
- 最終的に WordPress (sekkotsuin-komaki.com) へ反映予定
